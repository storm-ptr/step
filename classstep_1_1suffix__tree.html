Detailed Description

template<class T = char, class Size = size_t, template< class... > class Map = std::unordered_map>
class step::suffix_tree< T, Size, Map >

Ukkonen's online algorithm for constructing suffix tree.

Time complexity O(N*log(K)), space complexity O(N), where: N - text length, K - alphabet size.

Parameters
T- type of the characters;
Size- to specify the maximum number / offset of characters;
Map- to associate characters with edges, its key_type shall be T. std::map is preferable for small alphabet.
See also
https://en.wikipedia.org/wiki/Suffix_tree
