Detailed Description

template<class T = char, class Size = size_t, class Compare = std::less<>>
class step::suffix_array< T, Size, Compare >

Manber's algorithm for constructing suffix array.

Time complexity O(N*log(N)*log(N)), space complexity O(N), where: N - text length.

Parameters
T- type of the characters;
Size- to specify the maximum number / offset of characters;
Compare- to determine the order of characters.
See also
https://en.wikipedia.org/wiki/Suffix_array
